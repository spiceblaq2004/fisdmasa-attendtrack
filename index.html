Streamlined Workflow

1. Lecturer Action

Lecturer generates unique QR code at the start of each class session through their dashboard.

2. Student Scan

Students scan the displayed QR code using their mobile devices to register attendance instantly.

3. Automatic Recording

System automatically records attendance with timestamp and verifies student eligibility.

4. Real-time Analytics

Live dashboards provide immediate insights into attendance patterns and student engagement.
